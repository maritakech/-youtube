Задание1
Кейс №1. Клиент X хочет вывести на рынок новый товар. Он хочет использовать видеорекламу. Что мы можем ему предложить?
Так как это новый продукт основная цель видеоролика это запомниться пользователеям. Для этого используем бренд-метрику-Brand Lift. KPI-отчет post-campaign

Кейс №2. Клиент X хочет увеличить количество продаж. Он хочет использовать видеорекламу. Что мы можем ему предложить?
Основная цель данного клиента- это продажи, поэтому для данного кейса лучше использовать перформ-метрики. Для оценки эффективности используем показатели CPS, CPO,ДРР

Кейс №3. Клиент X хочет увеличить лояльность аудитории. Он хочет использовать видеорекламу. Что мы можем ему предложить?
Для увеличения лояльности аудитории можем также использовать бренд-метрику. Для оценки эффективности используем Brand Lift

Задание2
1) Показ по РК 1 => Переход из Instagram => Показ по РК 2 => Переход из поиска Google => Покупка
Конверсия будет атрибцуирована к показу по РК 2, так как пользователь увидел рекламу, но не перешел на сайт по ней, а вернулся через поиск Google и осуществил покупку

2) Показ по РК 1 => Переход с другого сайта => Клик по РК 2 => Переход по ссылке сайта => Покупка
Конверсия будет атрибуцирована Клику по РК 2, так как пользователь перешел по рекламе и осуществил покупку
3) Клик по РК 1 => Переход по закладкам => Клик по РК 2 => Показ по РК 1 => Покупка
Ценность присваивается последнему клику, поэтому конверсия будет атрибуцирована по по клику по РК 2

Задание3
Кейс №1. Клиент X хочет запустить видеорекламу, но ожидает, что стоимость продажи будет на уровне с баннерами или выше буквально на 10%. Такое возможно? Что можно ответить клиенту?
Доля видеорекламы растет, в то время как баннеры остаются на том же месте. Но видео может не принести желаемой конверсии, так как таргетинг в основном нацелен на холодную аудиторию.

Кейс №2. Клиент X - стартап, хочет с самого начала запустить видеорекламу, а только потом использовать другие виды рекламы. Стоит ли так делать? Что можно ему порекомендовать?
На мой взгляд рискованно запускать видеорекламу на стартапе, так как видеореклама больше подходит для больших компаний, которые уже знакомы на ранке

Кейс №3. Клиент X не видит смысла в видеорекламе. Он считает, что так как нет продаж по last-click, то она неэффективна. Что можно ответить?
В данном случае будут продажи по post-view или post-click. Конверсия присваивается последнему клику, выполненному пользователем перед покупкой или конверсией. Если не было кликов, конверсия присваивается последнему показу
